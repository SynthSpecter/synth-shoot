Synth Shoot

Bienvenue dans Synth Shoot, un jeu de tir spatial rétro-futuriste !

Comment jouer

  • Utilisez les flèches gauche et droite pour vous déplacer
  • Appuyez sur la barre d'espace pour tirer
  • Évitez les ennemis et collectez les power-ups

Objectifs

  • Détruisez autant d'ennemis que possible
  • Survivez le plus longtemps possible
  • Obtenez le meilleur score !
